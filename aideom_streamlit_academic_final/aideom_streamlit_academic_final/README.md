# AIDEOM-VN Streamlit Dashboard — Academic Final

Bản Streamlit được thiết kế lại theo phong cách website HTML mẫu: sidebar học thuật, nền giấy, card kết quả, tab tương tác, biểu đồ Plotly và hàm ý chính sách cho 12 bài.

## Chạy local
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Cấu trúc
- `app.py`: dashboard 12 bài AIDEOM-VN
- `data/`: 3 bảng dữ liệu Việt Nam dùng trong mô hình
- `.streamlit/config.toml`: theme sáng, màu chữ tương phản
