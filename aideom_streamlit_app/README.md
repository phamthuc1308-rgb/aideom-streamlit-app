# AIDEOM-VN Streamlit Dashboard

## Chạy local
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Đẩy lên Streamlit Community Cloud
1. Tạo repository GitHub mới.
2. Upload `app.py` và `requirements.txt`.
3. Vào https://share.streamlit.io/ hoặc Streamlit Community Cloud.
4. Chọn repo, branch, file chính là `app.py`.
5. Deploy.

## Ghi chú học thuật
Dashboard này được chuyển từ notebook AIDEOM-VN để trình bày kết quả mô hình ra quyết định. Khi nộp bài, nên nộp kèm notebook gốc và phần diễn giải chính sách.
