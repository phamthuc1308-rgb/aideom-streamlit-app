# AIDEOM-VN Interactive Streamlit Dashboard

Dashboard tương tác cho 12 bài của đồ án **Mô hình ra quyết định phát triển kinh tế Việt Nam trong kỉ nguyên AI**.

## Chạy local

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy Streamlit Cloud

- Repository: repo GitHub của bạn
- Branch: `main`
- Main file path: `app.py`

## Nội dung cải tiến

- Giao diện sidebar + hero cards theo phong cách premium.
- Tất cả bài 1–12 có tương tác bằng slider/input/data editor.
- Biểu đồ Plotly có hover, zoom, filter.
- Có bảng dữ liệu, KPI, kết quả tối ưu và diễn giải chính sách.
